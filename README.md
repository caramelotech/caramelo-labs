# 🧪 Caramelo Tech Labs

Laboratório prático da **Caramelo Tech** com foco em aprendizado hands-on através de exemplos, exercícios e projetos.

## 📚 O que você vai encontrar

* 📖 Anotações em Markdown
* 💻 Exemplos de código
* 🧪 Exercícios práticos
* 🚀 Mini projetos

## 🗂 Estrutura do repositório

```
/notes        → Anotações e estudos
/examples     → Exemplos simples
/exercises    → Exercícios propostos
/projects     → Projetos práticos
```

## 🚀 Como usar

1. Comece pelas anotações em `/notes`
2. Explore os exemplos em `/examples`
3. Resolva os exercícios em `/exercises`
4. Construa os projetos em `/projects`
